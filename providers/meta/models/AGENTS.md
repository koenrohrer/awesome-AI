# Agent Guide: Meta Models

This directory contains individual Llama and Meta safeguard model pages.

## Rules

- Cite exact HuggingFace model cards for context, modalities, license, and safety claims.
- Keep deployment and prompt-format caveats model-specific.
- Note when a model is a classifier or safeguard rather than a general assistant model.
- Use tables by default for comparable model facts and comparisons.
- Include benchmark tables only when the benchmark has a named source, date, and methodology.
