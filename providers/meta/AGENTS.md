# Agent Guide: Meta

This directory covers Llama-family open weights and adjacent Meta safety models.

## Rules

- Verify current public artifacts against the official Meta Llama HuggingFace organization and Meta docs.
- Check the exact Llama Community License terms for each model before commercial-use claims.
- Distinguish base models, multimodal models, and safeguard models.
- Do not imply Meta provides a first-party hosted API unless official docs say so.
