# Agent Guide: OpenAI

This directory covers OpenAI models and OpenAI-owned product surfaces.

## Rules

- Verify current model names, tool support, context windows, pricing, and availability against official OpenAI docs.
- Cite system cards, launch posts, and model pages for capability and safety claims.
- Keep Codex, ChatGPT Desktop, and image-generation surfaces separate when their workflows differ.
- Prefer Responses API guidance for new API-surface claims unless docs say otherwise.
