# Agent Guide: OpenAI Models

This directory contains individual OpenAI model-line pages.

## Rules

- Verify exact model IDs, supported tools, structured output support, pricing, context, and release status before editing.
- Cite OpenAI model pages and system cards directly.
- Keep Codex-optimized models distinct from general GPT models.
- Use tables by default for comparable model facts and comparisons.
- Include benchmark tables only when the benchmark has a named source, date, and methodology.
