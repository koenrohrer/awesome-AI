## What it is

GPT-5.3-Codex is OpenAI's dedicated agentic coding model for long-running coding, research, tool use, and complex execution workflows.

## Official system card in Markdown

This section is a markdown digest of the official OpenAI system card, not a verbatim transcription.

### Capability and product summary

- OpenAI describes GPT-5.3-Codex as its most capable agentic coding model at release time.
- It combines stronger coding performance with general reasoning and professional knowledge for long-running tasks.

### Safety and deployment summary

- OpenAI says GPT-5.3-Codex is treated as High capability in cybersecurity under the Preparedness Framework.
- The system card also says it is deployed with the corresponding safeguards used for high-capability cyber models.
- OpenAI notes High capability treatment on biology as well, while not reaching High on AI self-improvement.

### Integration notes

- Use GPT-5.3-Codex when you want the most coding-specialized OpenAI model rather than the broader GPT-5.4 family.
- If you need a generalist that absorbed much of Codex capability, compare against GPT-5.4.
