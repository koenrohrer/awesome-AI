# GPT-5.5 `[provider-doc]`

*Last reviewed: 2026-04-23.*

## At a glance

- Provider: OpenAI
- Model ID: `gpt-5.5` when API access launches
- Tier: Current product flagship for ChatGPT and Codex; API announced as coming soon
- Context window: 1M tokens announced for the upcoming API model; Codex product surface uses 400K context
- Official docs: [Introducing GPT-5.5](https://openai.com/index/introducing-gpt-5-5/), [GPT-5.5 system card](https://openai.com/index/gpt-5-5-system-card/)

## What it is

GPT-5.5 is OpenAI's newest model for complex real-world work, especially agentic coding, computer use, research, documents, spreadsheets, and tool-heavy workflows.

## Availability notes

- OpenAI says GPT-5.5 is rolling out to ChatGPT Plus, Pro, Business, and Enterprise users and to Codex users on Plus, Pro, Business, Enterprise, Edu, and Go plans.
- OpenAI says `gpt-5.5` will come to the API soon, but it was not launched to the API on April 23, 2026.
- The launch post announces planned API pricing and a 1M context window; verify against the live pricing and models pages before building around it.
