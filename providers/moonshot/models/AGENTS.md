# Agent Guide: Moonshot Models

## Rules

- Verify exact model IDs, thinking behavior, vision support, context, and pricing before editing.
- Cite Moonshot docs and HuggingFace model cards directly.
- Keep hosted API facts distinct from open-weight lineage claims.
- Use tables by default for comparable model facts and comparisons.
- Include benchmark tables only when the benchmark has a named source, date, and methodology.
