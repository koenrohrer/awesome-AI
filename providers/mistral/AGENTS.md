# Agent Guide: Mistral

This directory covers Mistral's proprietary platform and open-weight releases.

## Rules

- Verify whether each model is hosted-only, open-weight, or both.
- Cite Mistral platform docs, release posts, and HuggingFace model cards directly.
- Check the specific model license before making commercial-use claims.
- Note MoE memory and deployment implications for self-hosting.
