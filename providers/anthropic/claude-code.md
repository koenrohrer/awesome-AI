# Claude Code

Anthropic's terminal-native coding surface. This page exists separately from the Anthropic provider overview because Claude Code has its own runtime, permission model, extension surface, and workflow patterns.

## What belongs here

- Setup, authentication, and local runtime notes
- Claude Code-specific hooks, slash commands, and MCP usage
- Coding workflows that depend on terminal permissions, repo context, and local execution
- Automation patterns that are specific to Claude Code rather than generic shell or CI usage
