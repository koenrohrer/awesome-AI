# Agent Guide: Anthropic Models

This directory contains individual Claude model-line pages.

## Rules

- Verify exact model IDs, availability, retirement dates, context windows, and pricing before editing.
- Cite Anthropic model docs, deprecation docs, pricing, and system cards directly.
- Keep model-specific strengths and caveats distinct from general Claude provider guidance.
- Use tables by default for comparable model facts and comparisons.
- Include benchmark tables only when the benchmark has a named source, date, and methodology.
