## What it is

Claude Sonnet 4.6 is Anthropic's current balanced production tier. Anthropic positions it as the best speed/intelligence tradeoff across coding, computer use, long-context reasoning, and general professional work.

## Official system card in Markdown

This is a markdown digest of Anthropic's official system-card listing and release documentation, not a verbatim copy.

### Capability and product summary

- Sonnet 4.6 is the current default Sonnet line and a major upgrade over Sonnet 4.5.
- Anthropic describes improvements across coding, instruction following, agent planning, computer use, and long-context behavior.
- The current models overview lists Sonnet 4.6 with 1M context and 64K max output.

### Safety and deployment summary

- Anthropic lists a public system card for Claude Sonnet 4.6 on its system-card index.
- Anthropic's public messaging treats Sonnet 4.6 as the mainstream production choice rather than a special preview tier.

### Integration notes

- For most Anthropic API workloads, Sonnet 4.6 is the first model to try before jumping to Opus.
- Check the deprecations page if you are migrating from the older `claude-sonnet-4-20250514` snapshot.
