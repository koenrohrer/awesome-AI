# Agent Guide: Prompting

This is the flagship evidence-backed prompting directory.

## Evidence Bar

- Use `[paper]`, `[provider-doc]`, or `[tested]`.
- Do not add prompt folklore, social-media prompt libraries, or unsupported model-specific lore.
- Provider-specific prompting conventions belong in `providers/<provider>/` unless they are a general technique backed by evidence.

## Entry Guidance

- Explain the technique, when to use it, when not to use it, and what evidence supports it.
- Keep claims proportional to the cited result.
- Link tests from `prompting/tests/<slug>/` when granting `[tested]`.
- Update `prompting/README.md` when adding a page.
