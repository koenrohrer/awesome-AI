# Local LLM runners

Tools for running models on your own hardware: ollama, LMStudio, llama.cpp, vLLM, Jan, GPT4All, and similar.

## Inclusion bar

`[vetted-tool]` — maintained + used + scope-fit. See [CONTRIBUTING.md](../../CONTRIBUTING.md).
