# Agent Guide: Hardware

Hardware content must be sourced. The maintainer does not need to own the hardware for inclusion, but every claim needs a named source.

## Rules

- Use `[sourced]` only.
- Cite Apple, NVIDIA, AMD, Intel, HuggingFace, official benchmark pages, named videos, or reproducible public benchmarks.
- Include source names, dates, and timestamps where applicable.
- Do not include unnamed forum claims or "I heard" performance numbers.
