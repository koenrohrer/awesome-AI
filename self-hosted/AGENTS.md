# Agent Guide: Self-Hosted

This directory covers running AI locally or on owned infrastructure.

## Scope

- `runners/`: local model runners and serving stacks.
- `workflows/`: end-to-end local-first setups.
- `hardware/`: sourced hardware guidance.

## Rules

- Distinguish tool claims from hardware claims.
- For performance, compatibility, and cost claims, cite sources or maintainer-run tests.
- Avoid rumor-based setup advice.
