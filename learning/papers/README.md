# Papers

Landmark AI/LLM papers with a one-paragraph TL;DR and a link. Each TL;DR is written to be read *before* the paper, not instead of it.

## Inclusion bar

`[paper]` — full citation (authors, title, venue/year) and direct link to the paper (not a blog post about it). See [CONTRIBUTING.md](../../CONTRIBUTING.md).

## Scope

Papers that meaningfully changed practice — architectures, training recipes, reasoning techniques, agent patterns, eval methodology, safety. We don't try to mirror arxiv; we curate.
