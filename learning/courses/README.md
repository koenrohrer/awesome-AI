# Courses

Courses a maintainer completed and endorses. No "this looks good" — someone finished it and can speak to whether it was worth the time.

## Inclusion bar

Each course entry states:
- Provider (DeepLearning.AI, Coursera, fast.ai, university, etc.)
- Title
- Date a maintainer completed it
- One honest line on whether it was worth the time
