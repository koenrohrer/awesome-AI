# Agent Guide: Architecture

This directory links architecture explainers for foundational AI systems.

## Rules

- Prefer canonical, widely respected explainers over fresh commentary.
- Cover concepts such as transformers, MoE, diffusion, RLHF, DPO, and retrieval architectures.
- State why the resource is worth a reader's time.
- Avoid duplicating paper summaries that belong in `learning/papers/`.
