# Agent Guide: Learning

This directory covers foundations, not tips.

## Scope

- `papers/`: landmark papers with direct citations.
- `courses/`: courses a maintainer completed and endorses.
- `architecture/`: definitive or near-definitive explainers.
- `safety/`: defensive red-teaming, prompt injection, and jailbreak-resistance material.

## Rules

- Do not add generic blog posts unless the local README explicitly allows a definitive explainer.
- Keep summaries educational and honest about limitations.
- Update `learning/README.md` when adding public entries.
