# Hooks

Lifecycle hooks that run custom logic before or after an AI tool's actions (Claude Code hooks, pre-/post-tool-use callbacks, `SessionStart` handlers, etc.).

Hooks are how you teach a tool to enforce project conventions, inject context, or block unsafe actions without modifying the tool itself.
